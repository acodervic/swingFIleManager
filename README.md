## hotKeys
- ctrl+t newTab
- ctrl+w close now selected Tab
- ctrl+v copy selected
- ctrl+x copy selected
- ctrl+c paste files/dirs
- enter enter in dir
- backspace put seleted dirs/files to trash(only linux)
- ctrl+alt+q back
- ctrl+alt+e go
